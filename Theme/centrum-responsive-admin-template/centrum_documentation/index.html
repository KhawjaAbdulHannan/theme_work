Centrum How To's

How to change Header Styles?

The Centrum header comes with 4 available background options. You can switch between them by adding or change a classname on the header tag, avaialble in each html file of the template. The available classes are 'style2', 'style3' & 'style4'. 

<header class='style2'></header>

How to switch between available Top Area Styles?

The Centrum comes with four available toparea styles. On each html file of the template you will find a div with id toparea.

<div id="toparea"></div>

By default this div gets the 'Decent Style'. if you want to switch to any other style other than 'Decent Style' you can add one from available classes. The available classes are 'style2', 'style3' & 'style4'. 

<div id="toparea" class="style2"></div>

How to change Sideheader Style?

The Centrum sideheader comes with 4 Sideheader styles. You can switch between them by adding or change a classname on <div class="theme-content">, avaialble in each html file of the template. The available classes are 'topmenu', 'contained-menu' & 'narrow-menu'. 

<div class="theme-content topmenu">

How to change Sidemenu Background?

The Centrum sideheader comes with 4 available background options. You can switch between them by adding or change a classname on the Centrum-sidemenu class, avaialble in each html file of the template. The available classes are 'coloured', 'gradient', 'dark' & 'dark dark2'. 

<aside class="Centrum-sidemenu active gradient" id="sidemenu"></aside>

How to Set Width, Height and Position of a Panel?

For Centrum grid structure, an amazing plugin GridStack is used. gridstack.js is a mobile-friendly Javascript library for dashboard layout and creation. Making a drag-and-drop, multi-column dashboard has never been easier. gridstack.js allows you to build draggable, responsive bootstrap v3-friendly layouts. It also works great with knockout.js, angular.js, ember.

By using GridStack, it becomes very easy for even a non technical user to handle a widget's size and position, as you only need to pass a few attributes to the main div.To understand how Centrum handles its panels sizes and positions you need to read the GridStack docs which will make your life easier to play with Centrum widgets.
